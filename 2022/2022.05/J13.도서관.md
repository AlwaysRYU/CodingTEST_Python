# 도서관
https://www.acmicpc.net/problem/1461

### 유형
- 답구하기
- 그리디

### 첫번째 도전
- 로직을 짜는 것에 실패했다.
- 양수와 음수가 혼재되어 있을 때의처리가 명확하지 않았다.

### 풀이
- 음수와, 양수를 따로 해주어야한다.
- 따로 해주고 큰것부터 M개 씩 꺼내서 M개중 제일 큰걸 * 2 더해준다.
- 제일 먼 거리 만큼은 총합에서 감하여 답을 구한다.
- 자꾸 이동거리를 단위로 생각한게아니라 책을단위로 생각해서 로직을 짜지 못했다.
```java
package Y2022D05;

import java.io.BufferedReader;
import java.io.BufferedWriter;
import java.io.InputStreamReader;
import java.io.OutputStreamWriter;
import java.util.PriorityQueue;
import java.util.StringTokenizer;

public class J18_도서가ㅗㄴ3 {

	static BufferedWriter bw;
	public static void main(String[] args) throws Exception {
		BufferedReader br = new BufferedReader(new InputStreamReader(System.in));
		StringTokenizer st = null;
		bw = new BufferedWriter(new OutputStreamWriter(System.out));
		
		st = new StringTokenizer(br.readLine());
		int N = Integer.parseInt(st.nextToken());
        int M = Integer.parseInt(st.nextToken());

        // 두 우선순위 큐는 내림차순 정렬.
        PriorityQueue<Integer> positiveQ = new PriorityQueue<>((p1, p2) -> p2 - p1);
        PriorityQueue<Integer> negativeQ = new PriorityQueue<>((p1, p2) -> p2 - p1);

        st = new StringTokenizer(br.readLine());
        for (int i = 0; i < N; i++) {
            int temp = Integer.parseInt(st.nextToken());

            if (temp > 0) {
                positiveQ.offer(temp);
            } else {
                negativeQ.offer(Math.abs(temp));
            }
        }

        // 가장 멀리 있는 책의 위치를 저장.
        int maxV = 0;
        if (positiveQ.isEmpty()) {
            maxV = negativeQ.peek();
        } else if (negativeQ.isEmpty()) {
            maxV = positiveQ.peek();
        } else {
            maxV = Math.max(positiveQ.peek(), negativeQ.peek());
        }

        int ans = 0;
        

        while (!positiveQ.isEmpty()) {
            int temp = positiveQ.poll();
            for (int i = 0; i < M - 1; i++) {
                positiveQ.poll();

                if (positiveQ.isEmpty()) {
                    break;
                }
            }
            ans += temp * 2;
        }
        System.out.println(ans);
        
        while (!negativeQ.isEmpty()) {
            int temp = negativeQ.poll();
            for (int i = 0; i < M - 1; i++) {
                negativeQ.poll();

                if (negativeQ.isEmpty()) {
                    break;
                }
            }
            ans += temp * 2;
        }
        System.out.println(ans);
        ans -= maxV;
        bw.write(ans + "\n");
        bw.flush();
        bw.close();
        br.close();
    }
	
	

}
```
